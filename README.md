# Alpha-Connect-Four

A from-scratch [AlphaZero](https://www.science.org/doi/10.1126/science.aar6404) implementation in PyTorch that learns to play **Connect Four** and **Tic-Tac-Toe** purely through self-play, with no human game data or hand-crafted heuristics.

The agent combines Monte Carlo Tree Search (MCTS) with a residual convolutional neural network that jointly predicts a move policy and a position value. The network is trained on data generated by the agent playing against itself.

## How it works

The notebook is organized into a few self-contained components:

- **Game environments** — `TicTacToe` (3×3) and `ConnectFour` (6×7). Each exposes a common interface (`get_next_state`, `get_valid_moves`, `check_win`, `get_value_and_terminated`, `change_perspective`, `get_encoded_state`) so the rest of the code is game-agnostic. States are encoded as 3-plane tensors (opponent / empty / current player).
- **`ResNet`** — a convolutional residual network with a shared backbone and two heads: a **policy head** (move probabilities) and a **value head** (expected outcome in `[-1, 1]` via `tanh`). Depth and width are configurable (`num_resBlocks`, `num_hidden`).
- **`MCTS` / `MCTSParallel`** — neural-network-guided Monte Carlo Tree Search using PUCT selection, with Dirichlet noise added at the root for exploration. The parallel variant batches many self-play games through the network at once for speed.
- **`AlphaZero` / `AlphaZeroParallel`** — the training loop: self-play generates `(state, policy, outcome)` samples, then the network is trained to match MCTS visit-count policies (cross-entropy) and game outcomes (MSE). Checkpoints are saved per iteration.
- **`KaggleAgent`** — wraps a trained model as an agent compatible with [`kaggle_environments`](https://github.com/Kaggle/kaggle-environments) for evaluation.

## Requirements

Developed against:

- Python 3.10
- PyTorch 2.0.1 (CUDA 11.7 build; runs on CPU if no GPU is available)
- NumPy 1.23.5
- `matplotlib`, `tqdm`
- `kaggle_environments` (only for the evaluation cells)

```bash
pip install torch numpy matplotlib tqdm kaggle-environments
```

A CUDA-capable GPU is recommended for training but not required — the code falls back to CPU automatically.

## Usage

Open the notebook:

```bash
jupyter notebook AlphaZero.ipynb
```

- **Play against a trained Connect Four model:** run the cells up to and including the interactive game loop (it loads `model_2_ConnectFour.pt`). Enter a column index `0–6` on your turn; the AlphaZero agent responds via MCTS.
- **Train from scratch:** configure the `args` dictionary (search count, iterations, self-play games, batch size, etc.), instantiate a game and `ResNet`, and call `AlphaZeroParallel(...).learn()`. New checkpoints are written as `model_{iteration}_{game}.pt`.
